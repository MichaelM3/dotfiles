# Caveman Help

Available local Caveman equivalents:

- `caveman`: persistent terse response mode.
- `caveman-review`: compact line-level review findings.
- `caveman-commit`: compact conventional commit messages.
- `caveman-compress`: compress prose files safely.
- `caveman-stats`: report compression savings.
- `cavecrew`: delegation patterns for compressed subagent handoffs.

Activation phrases:

- `use caveman`
- `caveman mode`
- `caveman review`
- `caveman commit`
- `compress this`
- `use cavecrew`

Stop persistent terse mode with `stop caveman` or `normal mode`.
